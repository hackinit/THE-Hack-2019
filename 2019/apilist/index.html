Open API List

此列表的目的旨在与给参赛人员提供开发的idea与实现的工具, 以下提供的网址均有一些免费的API提供, 并可以在很短的时间申请到Auth Key

此列表是网络搜索与整合的结果, 但是网址所列出的api均没有测试过实际效果, 所以可能出现服务不可用, 或是用国外api要翻墙的情况. 建议参赛人员可以提前开始准备.

ps: 评价非常主观, 仅供参考
